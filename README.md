# Video Editing Landing Page

This is a landing page project built as part of my front-end development learning journey.

The goal of this project is to practice layout structuring, component building, and UI design using HTML and CSS.

## Tech Stack

* HTML5
* CSS3 (Flexbox)

## Current Features

* Header navigation
* Announcement bar
* Hero section
* Features section
